# Projet Strava

## Description

Ce projet est une application conçue pour interagir avec les données Strava. Elle permet de récupérer, analyser et visualiser les données d'activités sportives des utilisateurs. L'objectif est de fournir des outils supplémentaires pour exploiter les données Strava de manière personnalisée.

## Fonctionnalités

- Analyse des performances sportives (distance, temps, vitesse, etc.).
- Visualisation des données sous forme de graphiques et tableaux.
- Export des données pour un usage externe.

## Prérequis

- Un compte Strava avec une application enregistrée pour obtenir les données.
- Télécharger toutes les données des activitées strava et garder seulement le dossier activities et les fichiers activities et reactions

## Utilisation

1. Lancez l'application :
   Cliquez sur StravaApp

2. Actualisez la visualisation powerBI pour avoir les dernières données
